# com.valvesoftware.Steam.Utility.Spout2PW

Flatpak for [Spout2PW](https://github.com/hoshinolina/spout2pw).

Spout2PW is an extension for Proton/Wine that bridges [Spout2](https://spout.zeal.co) video streams into [PipeWire](https://www.pipewire.org) video streams.

You can use [OBS-PWVideo](https://github.com/flathub/com.obsproject.Studio.Plugin.OBSPWVideo) to add them as a source directly in the Linux version of OBS.

## Install

This flatpak is an extension for Steam and is therefore not listed on the Flathub website.

To install this plugin you need to run this command in a terminal:

```bash
flatpak install com.valvesoftware.Steam.Utility.Spout2PW
```

## Usage

Add in your Steam Launch Option:

```bash
spout2pw %command%
```

[Read the Spout2pw Wiki page for more info](https://github.com/hoshinolina/spout2pw/wiki)

## Uninstall

To install this plugin you need to run this command in a terminal:

```bash
flatpak uninstall com.valvesoftware.Steam.Utility.Spout2PW
```
